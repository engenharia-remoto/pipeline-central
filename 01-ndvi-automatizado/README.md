O processamento geoespacial para análise de imóveis rurais utilizando Google Earth Engine, Sentinel-2, NDVI e integração com bases vetoriais.
## Objetivos
- Automatizar o cálculo de NDVI por imóvel rural.
- Consultar imagens Sentinel-2 disponíveis em datas específicas.
- Gerar produtos para análise visual da vegetação.
- Apoiar a identificação de áreas irrigadas e pivôs centrais.
- Integrar resultados com fluxos de trabalho em QGIS.
## Requisitos - Python 3.10+ - Google Earth Engine 
- GeoPandas 
- PyYAML 
- Requests 
- Pillow 
- Matplotlib -
TQDM
